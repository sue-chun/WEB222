HTML5

This site utilizes HTML5: a markup language used for structuring and presenting content on the World Wide Web. It is the 5th version of HTML and the most current one. Older versions of HTML required plugins and APIs which are now automatically included in the one common interface of HTML5 for loading elements more efficiently. HTML5 also now includes features for multimedia and images as well as other features for structuring a page. As well, some elements have been removed for HTML5 and now use CSS to provide these functions instead such as font and center.
